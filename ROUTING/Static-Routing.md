
# Static Routing 
This is the type of routing where the routes are configured by the admnistrator manually to a specific path
meaning e.i the admin decide which path(route) the networking shall take to reach its destination
## Types of static routing 
*    Standard static routing
*    default static routing
*    Summary static routing
*    Floating static routing
